# nelloho.github.io

This is the source for my personal website, hosted on GitHub Pages.

## How to use

1.  Initialize a git repository in this directory:
    ```bash
    git init
    ```
2.  Add all the files to the repository:
    ```bash
    git add .
    ```
3.  Commit the files:
    ```bash
    git commit -m "Initial commit"
    ```
4.  Create a new repository on GitHub named `nelloho.github.io`.
5.  Add the remote repository:
    ```bash
    git remote add origin https://github.com/nelloho/nelloho.github.io.git
    ```
6.  Push the files to the repository:
    ```bash
    git push -u origin main
    ```

After a few minutes, your website should be live at `https://nelloho.github.io`.
